Esse projeto implementa classes para representar formas geométricas simples em um ambiente de programação orientada a objetos. As classes incluem ponto, linha, poligono, quadrado e circulo, 
com herança e encapsulamento para representar suas características e funcionalidades. Cada classe possui métodos para calcular características específicas, como área e perímetro. 
O código permite criar e trabalhar com diferentes formas.
